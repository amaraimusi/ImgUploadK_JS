# ImgUploadK_JS
Expand function of file upload element. (input type="file")

## JP
ファイルアップロード要素(input type="file")の機能を拡張する。

## Function
* extendDnD	:Extend drag & drop
* extendMultiple	:Extend multiple file upload function.
* extendLabel	:Extend file upload with label.
